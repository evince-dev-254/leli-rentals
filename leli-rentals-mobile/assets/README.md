# Assets Directory

This directory contains all the static assets for the Leli Rentals mobile app.

## Required Assets

To complete the app setup, you'll need to add the following files:

### App Icons
- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (32x32)

### Splash Screen
- `splash.png` - Splash screen image (1284x2778 for iPhone)

### Images
You can use the following placeholder images or replace with your own:
- Logo images (light and dark variants)
- Category images
- Listing images
- User avatars

## Image Sources

The app currently uses Unsplash images as placeholders. For production, you should:
1. Replace all placeholder images with your own assets
2. Optimize images for mobile performance
3. Ensure proper licensing for all images
4. Consider using a CDN for image delivery

## Optimization Tips

- Use WebP format for better compression
- Provide multiple resolutions for different screen densities
- Implement lazy loading for better performance
- Consider using Expo's Image component for automatic optimization
