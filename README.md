# Leli Rentals - Main Application

This is the main user-facing application for Leli Rentals.

## Getting Started

1.  Navigate to this directory:
    ```bash
    cd leli-rentals
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Features

- User authentication (Clerk)
- Property listings
- Booking system
- User profiles
